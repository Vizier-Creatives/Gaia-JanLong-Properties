🌿 Gaia-Jan Long Properties – Sustainable Real Estate Website

**Live Site:** [https://vizier-creatives.github.io/Gaia-Properties/](https://vizier-creatives.github.io/Gaia-Properties/)

Gaia-Jan Long Properties is a multi‑page, fully responsive website designed to showcase eco‑friendly real estate listings, company services, and a blog. The site emphasizes clean design, green aesthetics, and accessibility.

---

## 📁 Project Structure

```

Gaia-Jan Long Properties/
├── index.html        # Homepage with hero section and feature highlights
├── properties.html   # Listing of sample sustainable properties
├── services.html     # Services offered (sales, rental, green consulting)
├── about.html        # Company background and mission
├── blog.html         # Sample articles on sustainable living
└── contact.html      # Contact form and information
└── styles.css        # Unified styling – responsive, green‑themed

```

---

## 🚀 Viewing the Site

### Live on GitHub Pages
The site is automatically deployed from the `main` branch.  
Visit: [https://vizier-creatives.github.io/Gaia-Properties/](https://vizier-creatives.github.io/Gaia-Properties/)

### Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/vizier-creatives/Gaia-Jan Long Properties.git
```

1. Open index.html in any modern browser.

No build steps or dependencies required.

🎨 Design & Features

· Green & clean colour palette (#2c5f2d primary, white/light grey backgrounds)
· Fully responsive – works on desktop, tablet, and mobile
· Consistent navigation across all pages
· Interactive contact form (front‑end only – can be extended with backend)
· Card‑based property listings for easy scanning
· Blog previews to support content marketing

---

🛠️ Technologies Used

· HTML5
· CSS3 (Flexbox, Grid‑like layout, media queries)
· GitHub Pages for hosting
